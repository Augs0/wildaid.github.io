If you want to contribute to the O-FISH application, find an issue to solve:
  1. Android mobile app issues
  2. iOS mobile app issues
  3. Node.js web app issues
You will need to fork the appropriate code repository (Android, iOS, Web) to code your fix.

You will need to build the O-FISH application to test your fix.

Once you verify the issue is fixed, you will need to submit your patch to the github code. If you have built the O-FISH application and want to contribute, connect with us on the MongoDB Community Forum for Open Source Projects. We do not have exact contribution instructions right now, but we are developing them shortly, along with github issues for open issues that folks can easily use to contribute.
