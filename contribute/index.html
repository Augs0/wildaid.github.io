If you want to contribute to the O-FISH application, find an issue to solve:

  1. Android mobile app issues
  2. iOS mobile app issues
  3. React web app issues
  4. Realm backend

You will need to fork the appropriate code repository (Android, iOS, Web, Realm). You then need to checkout your new fork. For example:

git clone https://github.com/my-github-name/o-fish-web.git cd o-fish-web

Track changes from the original repo:

git remote add --track main upstream https://github.com/WildAid/o-fish-web.git git fetch upstream

Create and switch to a new branch on your fork:

git checkout -b fix_issue_12345 upstream/main

Code your fix and build the O-FISH application to test that it works.

Once you verify the issue is fixed, you will need to submit your edits to your fork's branch, commit, and push, e.g.:

git add . git commit -m 'fixing issue 12345' git push -u origin fix_issue_12345

Create a pull request as the output instructs by following the instructions given in the output from the previous command:

remote: Create a pull request for 'fix_issue_12345' on GitHub by visiting: remote: https://github.com/my-github-name/o-fish-web/pull/new/fix_issue_12345

A developer will review your code, and once the code is ready it will be merged in.

If you have any questions, connect with us on the MongoDB Community Forum for Open Source Projects.
